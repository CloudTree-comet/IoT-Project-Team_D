Lim Minseok _ Hardware & Framework Architect

Foundational Skeleton

We bridge the gap between simulation and reality using a modular skeleton designed for stability.

Hardware Wiring Skeleton

Implemented a structural skeleton connecting ESP32 with HX711, RTC, and LCD for seamless team integration.

Modular Framework (HAL)

Designed a Hardware Abstraction Layer (HAL) to allow 100% logic verification through virtual physics simulations.

Noise Isolation Strategy

Designed separate power rails for the 5V Servo and 3.3V MCU Logic to eliminate electrical interference during operation.

I2C Resource Optimization

Enabled the RTC and LCD to share the I2C bus (SDA/SCL), achieving high efficiency with minimal pin consumption.

Muhammad Hafiz Khalifa _ Firmware Core Logic

Deterministic Brain

The "Brain" manages complex feeding cycles and mission-critical safety protocols through real-time monitoring.

Intelligent monitorFeeding()

Engineered a core loop that calculates real-time weight delta, automatically stopping the motor when the target is reached.

Safety & Stuck Detection

Implemented jam detection and safety timeouts within the core logic to prevent mechanical failure during feeding.

NGO THANH DAT _ API & Web UI Lead

Cloud Synchronization

RESTful Endpoints

Developed handlers for /api/status and /api/manual-feed, allowing low-latency device control through JSON.

Real-Time Dashboard

Built a responsive interface using HTML, CSS, and JS to visualize weight logs and update schedules instantly.
